📖 MindSelf

Seu leitor pessoal de ebooks — modo escuro, marcações, anotações e progresso de leitura.


✨ Funcionalidades

📚 Biblioteca pessoal — importe seus próprios PDFs e arquivos TXT
🌑 Modo escuro — interface pensada para leitura confortável
🔖 Marcações de página — salve as páginas mais importantes
✏️ Anotações — escreva notas por livro e por página
📊 Progresso de leitura — acompanhe onde parou em cada livro
🔡 Fonte ajustável — tamanho e família de fonte personalizáveis
📱 Instala como app — funciona como aplicativo nativo no Android
💾 Funciona offline — seus livros e dados ficam salvos no dispositivo


📱 Como instalar no Android

Acesse o link do app no Chrome para Android:

   https://seuusuario.github.io/mindself

Toque nos três pontinhos ⋮ no canto superior direito
Selecione "Adicionar à tela inicial"
Pronto — o MindSelf aparece como ícone na sua tela! 🎉


🚀 Como usar

Toque em + para adicionar um livro (PDF ou TXT)
Toque na capa para abrir e começar a ler
Durante a leitura:

Toque na tela para mostrar/esconder os controles
Deslize para a esquerda/direita para trocar de página
Toque em 🔖 para marcar a página atual
Toque em ✏️ para adicionar uma anotação
Toque em ⚙️ para ajustar fonte e tamanho


Acesse suas marcações e notas pelo menu inferior


🗂️ Arquivos do projeto
mindself/
├── index.html      # App completo
├── manifest.json   # Configuração do PWA
└── sw.js           # Service Worker (modo offline)

🛠️ Tecnologias

HTML, CSS e JavaScript puro — sem dependências externas
PDF.js para renderização de PDFs
IndexedDB para armazenamento local dos livros
PWA (Progressive Web App) com suporte offline


📌 Observações

Os livros ficam salvos somente no seu dispositivo — nenhum dado é enviado para servidores
Compatível com Chrome para Android (recomendado) e navegadores modernos
Suporte a PDF e TXT — EPUB em breve
